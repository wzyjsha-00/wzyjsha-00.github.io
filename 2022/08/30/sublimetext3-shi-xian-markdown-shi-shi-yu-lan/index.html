Sublime Text3实现Markdown实时预览

发表于2022-08-30|更新于2022-08-30|软件教程
|阅读量:

MarkdomnPreview

利用快捷键在浏览器中预览Markdown文件，设置方法：

  1. 利用组合键Ctrl+Shift+P调出命令面板;
  2. 输入Package Control: Install Package,按下回车后在搜索框中输入MarkdomnPreview，等待安装成功；
  3. 在命令面板中输入Markdown Preview： Preview in Browser并选中，在出现的选项中任选github或markdown其一，其中：
    1. github是利用GitHub的在线API来解析.md文件，支持在线资源的预览；
    2. markdown就是传统的本地默认浏览器中显示预览结果，不支持在线资源的预览；
  4. 自定义预览快捷键，在Preferences -> Key Bindings打开的文件右侧栏的中括号中添加如下代码，其中alt+q可以设置为任意按键；
    { "keys": ["alt+q"], "command": "markdown_preview", "args": {"target": "browser", "parser":"markdown"} }

MarkdomnPreview + LiveReload实时刷新预览

设置方法：

  1. 参照1中方法安装MarkdownPreview插件，打开配置文件Preferences -> Package Settings -> Markdown Preview -> Settings，检查左侧enable_autoreload条目是否为true。若是则跳过，若不是，在右侧栏中添加下列代码后重启Sublime Text：
    {
    "enable_autoreload": true
}
  2. 安装LiveReload插件；
  3. 配置LiveReload插件：
    1. 利用组合键Ctrl+Shift+P调出命令面板，输入LiveReload: Enable/disable plug-ins, 按下回车后选择Simple Reload with delay (400ms)或者Simple Reload（没有延迟）；
    2. 打开配置文件Preferences-package Settings-liveReload-settings user,在右侧栏中添加下列代码后重启Sublime Text：
    3. {"enabled_plugins": ["SimpleReloadPlugin", "SimpleRefresh"]}
